# PRレビューチェックリスト

PRをマージする前に以下をすべて確認すること。

---

## 共通

- [ ] `docs/status.md` の対象タスク1件のみを変更している（複数タスクの同時実装は禁止）
- [ ] 不要なファイルを作成・残存させていない
- [ ] `.env.local` の値がコードや README に直書きされていない
- [ ] `pnpm lint` がエラーなしで完了する
- [ ] `pnpm build` が成功し `out/` が生成される

---

## Firebase

- [ ] Firestoreの読み書きが `users/{自分のuserId}` 配下のみに限定されている
- [ ] 他ユーザーのuidを直接指定したパスがない
- [ ] Firestoreセキュリティルールが `docs/requirements.md` の定義から緩くなっていない

---

## 論理削除・データ整合性

- [ ] `notes` / `links` / `boards` を物理削除（`delete()`等）するコードが無い
- [ ] 一覧表示・グループ化・木構造変換・CSV/JSON書き出しの処理で `isActive: true` の絞り込みが行われている
- [ ] メモを削除する処理で、関連する `links`（aまたはbがそのメモID）も合わせて `isActive: false` にしている
- [ ] 最後の1枚のボードを削除できないようになっている（UI・ロジック両方）

---

## リアルタイム同期

- [ ] 位置（x, y）・テキスト・削除・つながりの構造変更が `onSnapshot` で他デバイスに反映される
- [ ] パン位置・ズーム倍率・リスト画面の「根の指定」はローカル状態のみで、Firestoreに書き込まれていない
- [ ] カードのドラッグ中はFirestoreへの書き込みが発生せず、ドロップ時のみ書き込まれている

---

## Next.js

- [ ] `output: 'export'` に反するAPI Route / Server Actionを追加していない
- [ ] 動的ルート（`[id]`等）やURLへのID付与を追加していない（ボードID・ノートIDはURLに含めない方針）
- [ ] `'use client'` の宣言が必要なコンポーネントに付いている

---

## 認証

- [ ] 認証ガードが `/login` をリダイレクト対象から除外している
- [ ] 未認証時に保護されたページへアクセスすると `/login` へ遷移する
- [ ] サインアウト後に保護されたページへアクセスすると `/login` へ遷移する

---

## キャンバス操作

- [ ] 「切るモード」中はパン・カードドラッグ・つなぐ操作が無効化されている
- [ ] カードの展開アニメーション中、つながっている線（LinkLine）がカードの動きに追従している
- [ ] 展開アニメーションのイージングが `cubic-bezier(.16,.84,.44,1)` から変更されていない（変更する場合は要件定義書も更新する）

---

## UI・UX

- [ ] スマートフォン幅（375px）で操作できる
- [ ] タップ・クリック対象のボタンが小さすぎない
- [ ] エラー発生時に最低限のフィードバックがある（トーストなど）
- [ ] ローディング中に適切な表示がある
- [ ] 破壊的な操作（メモの一括削除・ボード削除）に確認ポップアップがある
